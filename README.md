#  Mini Projects

This repository contains small programming projects I build for practice and experimentation.

Most of the projects here are simple exercises meant to help me understand programming concepts better, try out ideas, and improve my problem-solving skills. They’re not large applications — just focused projects that helped me learn something new.

---

## Projects

### Tic-Tac-Toe

A simple implementation of the classic Tic-Tac-Toe game.
Built to practice game logic, condition checking, and basic program structure.

### Banking System (OOP Simulation)

A small object-oriented simulation of a banking system.
This project focuses on practicing OOP concepts such as:

* classes
* objects
* encapsulation
* account operations (deposit, withdraw, balance tracking)
